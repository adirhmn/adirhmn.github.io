Pada halaman informasi mengenai web ini, akan saya jelaskan beberapa kesulitan dalam pembuatan website ini.

3 Elemen Paling Sulit Untuk diimplementasikan

Dalam mengerjakan web ini ada tiga elemen paling sulit yang menurut saya untuk diimplementasikan, diantaranya:

1. Membuat tombol sign up dan sign in pada navbar

Pada saat pengerjaan web ini awalnya saya merasa kesulitann untuk membuat tombol di bagian navbar dan meletakkannya di bagian paling kanaan. Saya sudah mencoba untuk memasukkan class button ke dalam list item pada nav, namun hasilnya belum sesuai seperti ekspetasi saya.

2. Membuat tulisan dengan dua font dan warna yang berbeda pada satu kalimat

Mockup ini diambil pada bagian halaman sign up. Kesulitan yang saya alami adalah ketika harus membuat kalimat dengan dua warna yang berbeda dan dua jenis font yang berbeda.

3. Membuat bagian informasi leaders

Pada bagian informasi leaders ini, saya kesulitan untuk membuat bagian tengah dimana membuat dua kalimat pada satu column saling berbeda posisi. Untuk leader di bagian atas kiri, sedangkan untuk vice leader di bagian bawah kiri.

1 Desain Elemen Yang Impossible

Saat mengerjakan web ini, saya merasa sangat kesulitan untuk membuat logo dan navbar saling bedampingan di bagian kiri. jika menggunakan pure navbar dari materialize, maka logo dan navbar akan saling tumpang tindih.

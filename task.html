Cara Akses Member Area

Untuk mengakses member area, dapat mendaftar terlebib dahulu dengan menekan tombol sign up di menu. selanjutnya pergi ke halaman login. Pada username isi dengan code name dan password. untuk uji coba tanpa mendaftar, dapat dilakukan dengan mengisi username='member' dan password='member'

Cara Akses Admin Area

Untuk mengakses admin area, dapat dilakukan dengan pergi menuju halaman login. isi username='admin' dan password='admin'

List of user dan list of jobs dapat dilihat di admin area
